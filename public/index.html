📡 API Documentation
Use these endpoints to integrate LogicAI into your apps.
POST /api/chat
Send a text message and get a response.
curl -X POST /api/chat \ -H "Content-Type: application/json" \ -d '{"message": "Hello!", "session_id": "my-session"}'
POST /api/chat/upload
Send a message with a file attachment (multipart/form-data).
curl -X POST /api/chat/upload \ -F "message=Analyze this file" \ -F "session_id=my-session" \ -F "file=@document.pdf"
GET /api/history/:session_id
Retrieve conversation history for a session.
DEL /api/session/:session_id
Clear all messages in a session.
GET /api/health
Check if the server is running.
